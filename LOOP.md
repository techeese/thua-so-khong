# LOOP.md — the improve-and-ship loop's constitution

> Owner-approved 2026-07-02: **self-paced session loop · ship-then-veto · art in-bounds.**
> Scope: this repo (`thua-so-khong`) ONLY. The loop runs until the owner says stop.

## Invariants (never ship without)
1. `./gate.sh` green (band · syntax · fresh run · poisoned save) — band = **diagnosis > spreading > idling**.
2. `check.js` mirrors any `index.html` math change, same numbers.
3. Bilingual VI/EN for every player-facing string.
4. Thesis-safe: no great-man affordance, no lecturing (one line per beat), the multiplication stays the core.
5. Version bump in the footer + CHANGELOG entry + push + **live-URL verified**.

## The gradient
- **Owner feedback ALWAYS preempts everything** — anything the owner says jumps the queue.
- Otherwise rotate the compass (one item per iteration, small enough to ship):
  ART (Đông Hồ) → STORY/characters → FEEL/juice·audio → DEPTH/strategy → ENDINGS → MOBILE ergonomics → CONTENT → repeat.
- Every ~4th iteration is **BOLD** (new mechanic / character / structural experiment) — ships like everything else;
  the owner vetoes after playing (git revert is free).
- The loop cannot judge FUN. Each shipped iteration may leave the owner **at most one question**, never blocking.

## Halt-and-ask instead of inventing when
- gates fail twice on the same change · a change alters the game's identity (camera, thesis, cast, language)
- anything outward-facing beyond this existing repo/Pages site.

## Per-iteration protocol
1. **Orient**: LOOP.md → CHANGELOG (top) → ROADMAP → any new owner feedback.
2. **Pick ONE item** (feedback > roadmap > compass pointer). Keep the diff shippable in one iteration.
3. **Build** small; **verify**: gate.sh + a headless probe of the new path + screenshot self-review (~520px too).
4. **Ship**: version bump, CHANGELOG, commit (with session trailers), push, poll the live URL.
5. **Log**: ROADMAP checkboxes, memory file, move the compass pointer; note the next candidate.
6. **Schedule** the next wake (~25 min) with this same loop prompt.
